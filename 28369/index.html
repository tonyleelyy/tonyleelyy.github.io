Markdown 写作指南
发表于2020-04-20|更新于2020-04-20|教程

前言

想写这篇指南很久了。

在平时，这些情况你肯定不会少见：

  • 想发一份 .doc 文件给别人，在自己的电脑精心编辑了数遍，结果到了别人电脑的手里，不是打不开就是排版错乱……

  • 自己想写一篇文章，打开电脑上的 Word，先是调字体，接着调字号，调完了开始写文章，灵感已经少了一半。结果还没完，写的时候得调，写完之后还得调……

  • 用电脑写文章写到一半，得出门了。先是全篇复制，然后打开微信，发送给自己，自己在手机里再复制一遍，然后再粘贴到备忘录……最后在手机里写完还得重复一样的操作。

Markdown 能把这些问题完全解决。

  • 它极度标准化。在全世界，无论你使用哪种设备，几乎都使用同一种语法，同一种的规范。
  • 编辑器支持多种文件格式的转换。HTML、PDF、Doc 样样精通。
  • 因为它的标准化，你可以实现走到哪写到哪。手机写完平板写，平板写完电脑写。甚至你还可以用纸写，写完了输入编辑器，一键导出，完全不需要排版。
  • 可以说有了这个工具，你可以完全专注于文字写作。

使用软件

Windows & macOS & Linux:

  • 完全免费。官方网站：https://typora.io/

  • 界面简洁，自带实时生成效果，支持导出多种格式的文件，支持实时保存，还有多种主题样式支持。

  • 我这个网站里的所有文章都是使用它来创作的。

  • 最重要的是它支持 Windows，macOS 和 Linux 多种平台，使用体验无缝切换。

iOS & iPadOS:

MWeb

  • 免费下载，提供一次性内购。
  • 主题丰富，提供发布功能，使用方便。
  • 支持 macOS。

熊掌记

  • 免费下载，订阅制。
  • Apple 官方推荐，界面美观。
  • 支持 macOS。
  • 缺点：样式略显复杂。

Android:

纯纯写作

  • 免费下载，13块钱解锁完整版。
  • 号称永不丢失文件，可以实时保存。

使用方法

以下内容已经整理成了表格以供查阅：4. 表格整理

别看 Markdown 是一种特殊的写作语言，使用起来其实非常简单。

主要分为以下几个部分：

标题

Markdown 使用 # 标记标题，

一级标题为一个 #，二级为 ##，三级为 ###，以此类推

在写作的过程中，你只需要输入 # + 空格：

就能实现多级标题的输入。

符号后面都必须带有一个空格！

字体

Markdown 中实现 粗体 和 斜体 使用到 * 号

  • 粗体：
  • 斜体：
  • 高亮文字使用到 “`” 号

编号

在 Markdown 语法中，你可以使用 * 或 - 创建项目符号。

样例：

  • 富强
  • 民主
  • 文明
  • 和谐

在 Markdown 中，你可以使用 1. 2. 等创建编号。

样例：

  1. 我是第一项
  2. 我是第二项

符号后面都必须带有一个空格！

选框

你可以使用 - [ ] 来创建可以打勾的选框。

样例：

  • 未选择
  • 已选择

引用

你可以使用 > 大于号实现引用文字的效果。

样例：

我被引用了

跨设备方案

使用 iPhone + iPad + Windows + macOS

思路一

  • 使用 iCloud 服务同步。

  • 首先安装 Windows iCloud 客户端，实现 Windows 和 macOS 互联。

  • 手机端直接使用自带的 文件App，点击即打开 MWeb 进行编辑，实现自动保存和同步。

思路二

  • 使用 Windows SMB 服务同步。
  • 选中一个文件夹，右键 - 属性 - 共享 - 高级共享，勾选 “共享此文件夹”。
  • 使用 Documents App 连接电脑的 SMB 服务。
  • 启动对文件夹的同步。
  • 每次打开 Documents 即可实现同步。
